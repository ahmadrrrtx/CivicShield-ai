# Privacy

## Privacy approach
CivicShield AI is designed to minimize unsafe handling of sensitive information while supporting civic guidance workflows.

## Current behavior in code
- guest sessions supported
- authenticated sessions supported
- provider keys stored encrypted when persisted
- audit logs exist for governance actions
- reviewer access is scoped and gated

## Important user note
Users should avoid entering unnecessary sensitive personal information unless required for a legitimate guided workflow.

## Not yet fully complete
- full privacy policy review for production counsel
- data retention policy formalization
- subject access / deletion workflow automation
