# Roadmap

## Current release baseline
- Evidence-first civic AI chat
- Verified retrieval and citation pipeline
- Scoped settings persistence
- Conversation history and continuation
- Reviewer queue and audit tooling

## Near-term
- Email verification
- Password reset
- OAuth providers
- Live E2E automation
- Pagination and richer filtering for audit/reviewer surfaces

## Mid-term
- Database-backed role system beyond reviewer foundation
- Richer analytics and reporting
- CSV/export improvements
- Reviewer assignment and SLA workflows

## Long-term
- Multi-provider model routing
- Regionalized civic program catalogs
- Human escalation workflows
- Advanced compliance and audit tooling
