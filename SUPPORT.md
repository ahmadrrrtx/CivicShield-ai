# Support

## Getting help
Use repository issues for:
- bug reports
- feature requests
- documentation improvements

Use the deployment and launch runbooks first for setup questions:
- `DEPLOYMENT.md`
- `LAUNCH_READINESS.md`
- `scripts/production-env-checklist.md`
- `scripts/vercel-preview-verification-checklist.md`

## Before opening an issue
Please include:
- environment details
- reproduction steps
- expected behavior
- actual behavior
- relevant logs or screenshots
