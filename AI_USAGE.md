# AI Usage

## Provider model
- Current MVP provider: Groq
- Architecture supports future providers

## Trust controls
- retrieval runs before generation
- evidence is verified and filtered by policy
- citations remain visible in output
- fallback language is used when confidence is low

## Important limitations
- CivicShield AI explains but does not make legal or governmental decisions
- The assistant should not be treated as a substitute for official agency decisions
- Live provider behavior still depends on correct API and deployment configuration
